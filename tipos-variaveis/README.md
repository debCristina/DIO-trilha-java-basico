# dio-trilha-java-basico
# Tipos de variaveis em Java

Este é um projeto pessoal para praticar o uso e compreensão dos diferentes tipos de variáveis em Java. O objetivo é explorar os tipos primitivos e de referência, criando exemplos simples para testar e entender como funcionam em diversos cenários.

## Objetivo
O objetivo deste projeto é fornecer uma maneira prática de experimentar com diferentes tipos de variáveis em Java, incluindo:

Tipos Primitivos: Como `int`, `double`, `char`, `boolean`, entre outros.

Tipos de Referência: Como ``String, `arrays` e `objetos`.

## Estrutura do Projeto
Este repositório contém exemplos básicos de código que demonstram o uso dos tipos de variáveis em Java. Abaixo estão os principais tipos abordados:

**Tipos Primitivos**
- `int`: Números inteiros.
- `double`: Números decimais com precisão dupla.
- `char`: Caracteres únicos.
- `boolean`: Valores lógicos (true ou false).
- `byte`: Números inteiros pequenos.
- `short`: Números inteiros de tamanho reduzido.
- `long`: Números inteiros grandes.
- `float`: Números de ponto flutuante com precisão simples.

**Tipos de Referência**
- `String`: Cadeias de caracteres.
- `Arrays`: Coleções de elementos.
- `Objetos`: Como criar e manipular objetos de classes personalizadas.
